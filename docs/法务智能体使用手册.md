## 1. 合同审查

### 适用场景
- 收到对方发来的合作协议、推广协议、采购合同
- 需要审核我方起草的合同版本
- 对合同条款有疑问，需要风险分析

### 使用方法

发送合同文件（Word/PDF/文本），并说明需求：

> "审一下这份 CPS 推广协议"
> "这份合同的违约责任有没有问题？"
> "帮我看下第 5 条的知识产权条款"

### 多轮审核流程

**第 1 轮：我先读合同，针对性提问**

我会先读完整合同，然后问你几个关键问题（例如：违约金比例是否有预期？争议解决地是否便利？），同时让你选择审核强度。

**第 2 轮：按你选择的模式审核**

| 模式 | 适合场景 | 产出 |
|------|---------|------|
| 🔵 快速审查 | 只是扫一眼有没有大坑 | 仅标注致命风险，5-10分钟 |
| 🟡 标准审查 | 正常业务合同审核 | 逐条分析+四级风险评级 |
| 🔴 深度审查 | 高金额/复杂合同 | 三阶段工作流+法条校验+二次核验 |

### 风险等级说明

| 等级 | 含义 | 处理建议 |
|------|------|---------|
| 🔴 致命 | 合同无效/重大违约/刑事责任 | **必须修改** |
| 🟠 重要 | 权利义务显著失衡 | **建议修改** |
| 🟡 一般 | 表述不清/操作不便 | 可以修改 |
| 🔵 轻微 | 非核心问题，纯建议 | 仅供参考 |

### 审核报告包含
- 风险清单（逐条标注位置+风险等级+修改建议）
- 文本逻辑校验（主体验证/指代验证/逻辑验证/编号验证）
- 法律依据原文引用

## 2. 多合同对比

### 适用场景
- 对方修改了合同发回，想看看改了哪些地方
- 两家供应商的合同条款对比
- 新旧版本政策文件差异

### 使用方法

发送两份文档并说明对比需求：

> "对比一下这两份合同的区别"
> "看看对方改了我哪些条款"
> "帮我对比旧版和新版的分成比例变化"

### 对比报告包含

**摘要** — 修改/新增/删除的条款数量，高风险变更数量
**高风险变更** — 🔴标注的责任上限变更、终止条款修改、赔偿范围扩大等
**全部变更** — 按条款顺序逐条列出修改前→修改后内容
**未变更条款** — 无变化的条款列表

### 支持格式
- Word 文档（.docx）
- PDF
- 纯文本 / Markdown

## 3. 案件分析

### 适用场景
- 收到律师函或起诉状，需要评估
- 判断某个纠纷是否值得起诉
- 分析我方在某个法律问题上的立场

### 使用方法

描述具体情况：

> "玩家充值后要求退款，理由是未成年，怎么处理？"
> "有个推广渠道一直拖欠分成款，可以起诉吗？"
> "收到对方律师函说我们游戏侵权，怎么办？"

### 分析框架

| 阶段 | 内容 |
|------|------|
| 初步分析 | 案件类型识别 + 法律争议焦点 + 要件清单 |
| 法律校验 | 法条有效性检查 + 新旧法适用判断 |
| 反思修正 | 标记需更新法规 + 更新分析 + 最终意见 |

### 输出示例
- **案由定性**：属于什么类型的法律纠纷
- **法律依据**：适用的法条原文引用
- **风险评估**：胜诉可能性、赔偿区间、周期预估
- **证据清单**：需要准备哪些证据
- **行动建议**：下一步操作

## 4. 法律文书起草

### 适用场景
- 需要发律师函催款或通知
- 草拟起诉状准备立案
- 写保密协议、竞业限制协议等

### 使用方法

说出需求和关键信息：

> "帮我起草一份催款律师函，对方欠了 50 万分成长"
> "写一份起诉状，要求对方支付拖欠的推广费"
> "生成一份竞业限制协议"

### 可起草的文书类型

| 类型 | 常用文书 |
|------|---------|
| 诉前文书 | 律师函、催告函、解约函 |
| 诉讼文书 | 起诉状、证据目录、保全申请书 |
| 合同协议 | 保密协议、竞业限制协议、和解协议 |
| 投诉文书 | 著作权侵权投诉书、行政举报书 |

### 文书格式
- 正式司法文书格式
- 含当事人信息、案由、事实、理由、法律依据、请求事项
- 附参考例文原文

## 5. 诉讼节点管理

### 适用场景
- 案件进入诉讼程序，需要跟踪进度
- 不确定下一步该做什么
- 担心错过时效或期限

### 使用方法

告诉案件当前状态：

> "我们有个案子已经立案了，下一步是什么？"
> "对方起诉了我们，现在要准备什么材料？"
> "一审输了，上诉期限多久？"

### 诉讼流程参考

```
起诉 → 立案 → 送达 → 举证 → 开庭 → 判决 → 上诉/执行
  │       │       │       │       │       │
  管辖     7日    15日    30日   排期    15日
  确认    审查    答辩    举证         上诉期
```

### 输出包含
- 当前所处的诉讼阶段
- 下一步待办事项（附期限）
- 需要准备的证据材料
- 时间节点提醒

## 6. 策略建议

### 适用场景
- 需要在应诉和和解之间做决策
- 不确定某个法律争议的处理方向
- 想要评估不同方案的利弊

### 使用方法

描述争议场景和你的选项：

> "对方要求赔偿 100 万，我们应诉还是和解划算？"
> "这个纠纷走诉讼快还是仲裁快？"
> "对方愿意调解，给个谈判方案"

### 策略分析维度
- **法律风险评估**：胜诉/败诉可能性
- **成本分析**：诉讼费、律师费、时间成本
- **商业影响**：合作关系、行业声誉、业务连续性
- **可选方案**：谈判/调解/诉讼/仲裁
- **推荐策略**：基于综合评估的最优方案

## 7. 法律知识 FAQ

### 适用场景
- 日常法律疑问快速确认
- 不确定某个法律概念或规定
- 需要快速查法条

### 使用方法

直接提问：

> "广告法里极限词有哪些？"
> "私服运营者会判几年？"
> "竞业限制的补偿金标准是多少？"

### FAQ 覆盖范围
- 游戏行业专项（传奇IP、私服、外挂、盗版）
- 广告法合规（极限词、爆率宣传、概率公示）
- 消费者保护（退款、未成年人、欺诈）
- 劳动用工（离职、竞业、工伤）
- 合同法基础（违约、解除、赔偿）

## 8. 工伤指引

### 适用场景
- 员工发生工伤，不知道处理流程
- 需要判断是否属于工伤
- 计算工伤赔偿金额

### 使用方法

描述情况：

> "员工在上班路上发生交通事故，算工伤吗？"
> "工伤认定需要什么材料？"
> "十级伤残能赔多少钱？"

### 指引内容
- **工伤认定标准**：哪些情形算/不算工伤
- **处理流程**：送医→报告→认定→鉴定→赔偿
- **材料清单**：工伤认定申请所需材料
- **赔偿标准**：一次性伤残补助金计算
- **时效提醒**：用人单位 30 日内申报，职工 1 年内申报

## 通用说明

### 对话技巧

- **描述越具体，分析越精准**：合同金额、纠纷背景、双方关系
- **可以随时追问**：对分析有疑问，直接问"为什么""依据是什么"
- **可以打断纠正**：如果方向不对，说"不，情况是……"

### 格式支持

| 文件类型 | 是否支持 | 说明 |
|---------|---------|------|
| Word (.docx) | ✅ 支持 | 直接发送文件 |
| PDF（电子版） | ✅ 支持 | 直接发送文件 |
| 纯文本 | ✅ 支持 | 粘贴文本或直接描述 |
| 扫描件 PDF | ⚠️ 有限支持 | OCR 识别，建议提供可编辑版本 |
| 图片 | ❌ 不支持 | 请提供文本 |

### 引用规范

所有法律分析均附法律依据原文引用：
- 成文法 → 《XXX法》第X条 + 原文
- 司法解释 → 法释〔20XX〕XX号 第X条 + 原文
- 指导案例 → 案号/第X号 + 裁判要点

### 重要提示

> 本助手提供的是法律分析参考，**不构成正式律师意见**。
> 涉及重大合同签订、诉讼决策、大额赔偿，建议委托专业律师。
> 法律校验受限于公开信息源，关键条款建议人工复核。
