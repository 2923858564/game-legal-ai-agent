# 引用规范（员工版）

与内部法务智能体共享同一套引用标准，**不因能力弱化而降低标准**。

## 法可依原则

每条建议必须有法律依据引用：

| 引用类型 | 格式 | 示例 |
|----------|------|------|
| 成文法 | 《{法律名称》》第X条——"原文内容……" | 《民法典》第584条——"损失赔偿额应当相当于因违约所造成的损失……" |
| 司法解释 | 法释〔20XX〕XX号 第X条——"原文内容……" | 法释〔2023〕12号 第65条 |
| 行政法规 | 《{条例}》第X条——"原文内容……" | 《工伤保险条例》第14条 |

## 引用标注规则

| 情况 | 格式 |
|------|------|
| 已查到明确法条 | 法律名称 + 条文号 + **原文引用** |
| 仅有网上搜索结果 | `【法律依据】（网上搜索：{来源简述}）` |
| 完全未查到 | `【法律依据】（未查到直接引用的法例，以下为法律原则分析）` |

## 不可接受的做法

- 给出建议但不列法律依据
- 只写法条号不写原文
- 编造法条（使用不存在或已废止的法条）
- 模糊引用（"根据相关法律规定"）
- 改写概括法条而非原文引用

## 引用校验清单

- [ ] 每条建议是否有法律依据？
- [ ] 法条引用是否附完整原文？
- [ ] 法规名称 + 条文号是否准确？
- [ ] 引用来源是否标注？
