# 合同预审

## 定位

合同预审是**快速扫读**，不是逐条深度审核。发现明显风险即可，不做三阶段工作流、不做 IRAC 分析、不做六段式要件清单。

## 输入

用户上传的合同文本或粘贴的合同条款。

## 操作步骤

1. 快速阅读合同全文，识别合同类型和关键条款
2. 仅标注以下两类风险：
   - **🔴 致命风险**：合同无效、重大违约、刑事责任、核心权利义务严重失衡
   - **🟠 重要风险**：显著失衡、重大风险敞口、关键条款缺失
3. 给出简要修改建议
4. 末尾附带免责声明 + 发现 🔴 风险时提示转法务

## 输出格式

```
## 合同预审意见（快速扫读）

### 🔴 致命风险
- {条款位置}：{风险描述} → {修改建议}

### 🟠 重要风险
- {条款位置}：{风险描述} → {修改建议}

### 简要总结
{2-3 句话总体评价}

---
⚠️ 免责声明：此为AI辅助预审，非法务正式审核意见。
{如有 🔴 风险，追加：涉及重大权益，建议转法务部终审确认。}
```

## 风险评级标准

| 等级 | 含义 | 行动 |
|------|------|------|
| 🔴 致命 | 合同无效/重大违约/刑事责任/核心失衡 | 必须修改，提示转法务 |
| 🟠 重要 | 显著失衡/重大风险敞口 | 建议修改 |
| 其他 | 一般/轻微问题 | 预审阶段不标注，避免过度干预 |
