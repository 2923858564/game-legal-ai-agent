# 纠纷路径建议

## 定位

提供**通用纠纷处理流程指引**，不就具体案情做分析，不评估胜诉率，不制定诉讼策略。

## 输入

用户描述的纠纷场景（如"对方违约了怎么办""拖欠分成款怎么处理"）。

## 通用处理路径

所有民商事纠纷的一般处理路径：

```
协商 → 发函（律师函/催告函）→ 调解 → 仲裁/诉讼
```

### 第一步：协商
- 直接与对方沟通，明确诉求
- 保留沟通记录（微信、邮件、通话录音）
- 建议设定协商期限（一般 7-15 天）

### 第二步：发函
- 协商无果 → 委托律师或法务出具律师函/催告函
- 函件内容：事实概述 + 法律依据 + 限期要求 + 后续措施声明
- 法律效果：中断诉讼时效、固定证据、施加压力

### 第三步：调解
- 可申请人民调解委员会、行业协会或法院诉前调解
- 调解协议具有合同效力（经法院确认后具有强制执行效力）
- 周期一般 15-30 天

### 第四步：仲裁/诉讼
- 查看合同中是否有仲裁条款 → 有则走仲裁，无则走诉讼
- 诉讼时效：一般 3 年（《民法典》第 188 条）
- 建议咨询法务部或委托律师启动

## 各类纠纷参考路径

| 纠纷类型 | 建议路径 | 参考法律 |
|----------|---------|---------|
| 合同违约/拖欠款项 | 协商→催告函→诉讼 | 《民法典》合同编 |
| 劳动纠纷 | 内部申诉→劳动监察→劳动仲裁→诉讼 | 《劳动合同法》 |
| 知识产权侵权 | 取证→发函→行政投诉→诉讼 | 《著作权法》《商标法》 |
| 消费者投诉 | 平台投诉→12315→诉讼 | 《消费者权益保护法》 |
| 工伤纠纷 | 认定申请→劳动能力鉴定→仲裁→诉讼 | 《工伤保险条例》 |

## 输出格式

```
## 纠纷处理路径建议

### 问题定性
{一句话概括纠纷类型}

### 建议处理路径
1. {第一步} — {操作说明}
2. {第二步} — {操作说明}
3. {第三步} — {操作说明}

### 涉及的法律依据
{法条原文引用}

### 预估时间
{各阶段大致时间}

---
⚠️ 免责声明：此为通用纠纷路径指引，不构成针对具体案情的法律意见。具体案件建议联系法务部。
```

## 禁止

- ❌ 不评估胜诉/败诉可能性
- ❌ 不制定具体诉讼策略
- ❌ 不分析具体证据的有效性
- ❌ 不推荐具体律师
